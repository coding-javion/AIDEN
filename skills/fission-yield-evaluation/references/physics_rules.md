# 裂变产额物理规则

本文件存放 `fission-yield-evaluation` skill 的详细审计规则。使用前先确认元数据和规则适用域。

## 状态定义

- `PASS`：检查通过，证据支持数据合理。
- `WARN`：存在物理上可疑、经验上异常或元数据较弱的问题，但不足以判定硬错误。
- `FAIL`：违反明确硬约束，或在适用域内严重违反基础物理。
- `BLOCKER`：缺少关键元数据，导致用户请求的核心判断无法完成。
- `INSUFFICIENT_METADATA`：某条规则需要的元数据不足，不能应用。
- `NA`：规则不适用于该数据类型、能区或体系。
- `INFO`：有用观察，不构成问题。

## 评价层级

### 第 1 级：硬约束

明确违反时通常可给 `FAIL`，极轻程度的违反可给 `WARN`：

- 产额必须是有限数；
- 产额必须非负；
- 不确定度必须是有限数且非负；
- 协方差矩阵如存在，应在数值容差内对称且半正定；
- 同质异能态比例必须满足 `0 <= R_m <= 1`；
- 核素标识必须物理可识别，不能出现不可能的 `A`、`Z`、`N` 组合；
- 一次二分裂碎片对必须近似守恒：
  - `A_L + A_H ≈ A_F`；
  - `Z_L + Z_H ≈ Z_F`；
- 完整独立二碎片产物分布总和应约为 `2.0` 每次裂变，或 `200` 每 100 次裂变。

### 第 2 级：强经验系统学

这一级用于发现物理上可疑的数据形态。通常给 `WARN`，不要自动给 `FAIL`；只有当数据同时满足明确适用域、分布完整、偏离极大且没有任何元数据解释时，才考虑结合第 1 级硬约束或具体规则给更高严重性。

使用顺序：

1. 先应用普适或近普适规律，检查分布是否基本连续、定义是否一致、局部形态是否像物理产额。
2. 再根据裂变体系、能区、反应道和前/后中子约定，决定是否应用特定核区规律。
3. 若适用域不明，相关特定核区规律给 `INSUFFICIENT_METADATA` 或 `NA`，不要按默认低能锕系处理。

#### 2A：普适或近普适规律

这些规律不依赖某一个特定核区，但仍要求数据类型合适、分布足够完整。它们主要用于发现单位混用、核素标识错误、产额定义混用或局部异常点。

应检查：

- 定义一致性：同一张表或同一比较组内，产额类型、单位、归一化、前/后中子约定应一致；不同定义不能直接混合比较。
- 分布连续性：完整或近完整的 `Y(A)`、`Y(Z)`、`Y(A,Z)` 通常随相邻 `A` 或 `Z` 平滑变化；孤立尖峰、断崖式跳变或空洞需要用不确定度、阈值或物理结构解释。
- 固定质量链电荷包络：固定 `A` 下的 `Y(A,Z)` 通常围绕最概然电荷 `Z_p` 单峰分布；允许奇偶振荡，但主包络不应无解释地极宽、断裂或多峰。
- UCD 粗基线：`Z_UCD(A) = A * Z_F / A_F` 可作为电荷分布的粗参考，但 `Z_p` 可因电荷极化偏离 UCD；不要要求二者严格相等。
- 二分裂互补性：二分裂主通道的轻、重碎片在质量和电荷上应具有互补关系；若数据给出成对一次碎片，守恒检查属于第 1 级硬约束。
- 前/后中子约定：中子发射后产物质量数相对一次碎片会向低 `A` 移动；不能把前中子峰位和后中子峰位当成同一物理量直接比较。

典型 `WARN`：

- 相邻质量数或电荷数间出现远超不确定度的孤立跳变，且没有阈值、选择效应或壳结构说明。
- 固定 `A` 电荷链呈多个孤立主峰，或主产额集中在离 UCD 很远的区域且无解释。
- 同一数据集中疑似混入独立产额和累积产额，或混用每次裂变分数和每 100 次裂变百分数。

#### 2B：适用域特定核区规律

这些规律只在特定核区、能区和反应道下适用。使用前必须确认裂变体系、能量、产额阶段、产额类型和完整性。

低能锕系裂变：

- 适用对象：锕系自发裂变、热中子诱发裂变、低能中子诱发裂变，且 `Y(A)` 完整或近完整。
- 质量形态：通常呈非对称双峰；对称分裂附近 `A ≈ A_F / 2` 常为谷区。
- 重峰区域：重碎片峰常大致位于 `A ≈ 132-145`；常见结构在 `A ≈ 134` 和 `A ≈ 140-142` 附近。
- 壳结构：`Z = 50`、`N = 82` 及 `N ≈ 88` 附近形变重碎片区域可出现产额增强。
- 奇偶效应：低激发时偶 `Z`、偶 `N` 碎片常增强，但这是统计增强项，不是逐核素硬约束。

高能锕系裂变：

- 对称分裂分量通常增强，低能双峰结构可能变浅。
- 壳效应和奇偶振荡通常减弱。
- 若跨越多机会裂变阈值，峰位和峰谷比可能出现非平滑变化。

前锕系或中质量体系：

- 质量分布可能更接近对称，或呈与低能锕系不同的模式。
- 不应要求出现典型低能锕系双峰，也不应要求重峰落在 `A ≈ 132-145`。

奇异核、超重体系、重离子诱发裂变：

- 可能受入口道角动量、形变、准裂变、壳结构和有限统计显著影响。
- 低能锕系经验只能作为背景提示，通常不给强判据。

典型 `WARN`：

- 标称完整低能锕系 `Y(A)` 完全没有非对称双峰，或重峰无解释地偏离典型重碎片区域。
- 高能数据仍被按低能锕系强壳结构逐条判错。
- 前锕系、中质量或超重体系被错误套用低能锕系峰位规则。

### 第 3 级：能量与反应体系趋势

这一级用于比较不同能量、不同反应道或不同裂变体系之间的趋势。它比第 2 级更依赖元数据；如果能量、反应道、产额定义或有效裂变核不清楚，优先给 `INSUFFICIENT_METADATA`，不要强行判断。

使用顺序：

1. 先确认比较对象是否同规范：同一产额类型、同一归一化、同一前/后中子约定、同一完整性水平。
2. 再应用普适或近普适的激发能趋势。
3. 最后按核区、能区和反应机制套用特定趋势；跨越多机会裂变阈值时尤其要谨慎。

#### 3A：普适或近普适趋势

这些趋势主要反映激发能增加后壳结构、奇偶结构和局部分布细节被统计效应冲淡。只适用于同一裂变体系或可明确定义的同规范比较。

通常期望：

- 分布平滑化：激发能增加后，局部壳结构和细小起伏通常被削弱。
- 奇偶减弱：偶奇振荡通常随激发能增加而减弱。
- 宽度增加：可用能量增加通常使可访问碎片构型增多，质量或电荷分布可能变宽。
- 前/后中子差异增强：较高激发下瞬发中子发射更重要，后中子产物分布相对一次碎片的质量位移和展宽可能更明显。
- 规范不可混比：独立产额与累积产额、前中子与后中子、完整分布与阈值筛选子集不能直接比较能量趋势。

典型 `WARN`：

- 标称同一规范的多能量数据中，奇偶振荡或局部壳结构随能量升高反而系统增强，且无模型或实验说明。
- 高能数据比低能数据更尖锐、更窄，且不能由产额定义、选择阈值或多机会裂变解释。

#### 3B：适用域特定核区趋势

这些趋势只在特定核区或反应机制下成立。使用前必须确认裂变体系、入射能区和有效裂变核。

低能到中能锕系裂变：

- 激发能增加通常提高对称分裂区域产额。
- 峰谷比通常降低，非对称双峰结构变浅。
- 壳驱动的重峰结构通常减弱。
- 跨同位素比较时，重峰位置常比轻峰更稳定；轻峰随裂变核质量按互补性移动。

高能锕系裂变：

- 对称分量通常更重要，低能锕系峰位和峰谷比规则只能作为弱参考。
- 多机会裂变会使有效裂变核变轻，导致峰位、宽度和归一化趋势出现折点。

多机会裂变阈值附近：

- 入射粒子可能先发射一个或多个中子，再由较轻的有效复合核裂变。
- 趋势可以突变，不能按单一 `A_F` 连续外推。
- 需要检查元数据中的 `A_F` 是初始复合核，还是按裂变机会加权后的有效裂变核。
- 参考库或模型比较必须采用同样的多机会裂变处理。

前锕系、中质量、奇异核和超重体系：

- 质量分布、对称分量和壳效应随能量的变化可能不同于锕系。
- 重离子诱发裂变还可能受入口道角动量、形变和准裂变影响。
- 经验趋势通常只能给 `INFO` 或弱 `WARN`，除非有同体系同规范的清晰基线。

典型 `INFO`、`WARN` 或 `NA`：

- 趋势突变与多机会裂变区域一致时，给 `INFO`，不要当成异常。
- 数据跨越多机会裂变阈值却仍按单一裂变核解释，给 `WARN` 或 `INSUFFICIENT_METADATA`。
- 体系不是低能锕系时，低能锕系能量趋势规则通常为 `NA` 或弱 `INFO`。
- 反应道、能量、完整性或产额定义缺失时，不做能量趋势判断。

### 第 4 级：模型依赖或定性检查

谨慎使用，通常作为辅助证据：

- 高斯模式分解；
- Wahl 风格 `Z_p` 电荷系统学；
- UCD 加电荷极化比较；
- GEF、JENDL、ENDF/B、JEFF 等评价库比较；
- 瞬发中子锯齿形规律一致性；
- TKE 与瞬发中子多重数一致性；
- 基于自旋、宇称、能级密度和 gamma 级联的同质异能态比例判断。

## 详细规则清单

### META-001：声明产额类型

检查数据集是否清楚说明产额是独立、累积、质量链、元素、同位素还是同质异能态产额。

- `PASS`：清楚。
- `BLOCKER`：不清楚，且请求判断依赖该信息。
- `WARN`：可推断，但未明确记录。

### META-002：声明中子发射前/后

检查数据集对应中子发射前一次碎片还是中子发射后产物。

- `PASS`：清楚。
- `WARN`：可推断，但未明确说明。
- `BLOCKER`：不清楚，且质量守恒或峰位检查依赖该信息。

### META-003：声明裂变体系和能量

检查 `Z_F`、`A_F`、反应道和能量是否已知。

- `PASS`：体系和能量清楚。
- `WARN`：体系已知但能量含糊。
- `BLOCKER`：裂变体系未知。

### NUM-001：有限且非负的产额

检查无 `NaN`、无 `Inf`、无负产额。

- `FAIL`：存在负产额或非有限产额。
- `PASS`：否则。

### NUM-002：不确定度有效性

检查不确定度是否有限、非负，并确认主要产额的相对不确定度是否明显异常。

- `FAIL`：存在负不确定度或非有限不确定度。
- `WARN`：实验数据缺少不确定度，或主要产额相对不确定度极端异常且无说明。
- `INFO`：理论输出有意不提供不确定度。

### NUM-003：归一化

仅适用于完整独立产物分布。

期望总和：

- 每次裂变分数：约 `2.0`；
- 每 100 次裂变百分数：约 `200`。

不要用于累积同位素列表、部分测量集合、选定 gamma 计数同位素、阈值筛选数据、部分质量窗口、缺失大量低产额产物的独立产额。

- `FAIL`：完整独立数据严重归一化错误。
- `WARN`：中等偏差。
- `NA`：不适用。

### NUM-004：协方差有效性

如提供协方差矩阵，检查对称性、对角方差非负和半正定性。

- `FAIL`：存在较大负特征值、明显非对称或负方差。
- `WARN`：小的数值负特征值或轻微非对称。
- `NA`：无协方差。

### MASS-001：低能锕系双峰结构

适用于低能锕系裂变且 `Y(A)` 完整或近似完整。

期望：非对称双峰质量分布、对称分裂附近谷区、显著轻峰和重峰。

- `PASS`：清楚双峰形态。
- `WARN`：双峰较弱、缺失或异常对称。
- `FAIL`：完整热中子/低能锕系质量产额平坦或只有单个对称峰且无解释。
- `NA`：规则不适用。

### MASS-002：重峰位置

低能锕系裂变重峰大致应在 `A ≈ 132-145`，常见结构在 `A ≈ 134` 和 `A ≈ 140-142` 附近。

- `PASS`：峰位合理。
- `WARN`：重峰无解释地偏移数个质量单位。
- `NA`：高能、非锕系、稀疏或未知体系。

注意中子发射后产物会因瞬发中子发射向低质量数移动。

### MASS-003：互补性

对二分裂一次碎片对检查 `A_L + A_H ≈ A_F` 与 `Z_L + Z_H ≈ Z_F`。对中子发射后产物，质量数应低于对应一次碎片质量数。

- `FAIL`：一次碎片对违反守恒。
- `WARN`：中子发射前/后约定混用。
- `NA`：无成对碎片信息。

### MASS-004：能量趋势

多能量数据集应检查能量增加是否通常导致质量分布变宽、峰谷比降低、对称分量增加、壳效应和奇偶效应减弱。

- `PASS`：趋势大体合理。
- `WARN`：趋势与预期相反且无解释。
- `INFO`：突变与已知多机会裂变区域一致。
- `NA`：单能量数据集。

### CHARGE-001：固定 A 的电荷分布

对固定 `A` 下多个 `Z` 的 `Y(A,Z)`，允许奇偶效应后应围绕 `Z_p` 近似高斯。

- `PASS`：单峰且近似高斯。
- `WARN`：极宽、强烈偏移、不连续或无解释多峰。
- `NA`：每条质量链同位素数量不足。

### CHARGE-002：UCD 基线与电荷极化

使用不变电荷分布基线：

```text
Z_UCD(A) = A * Z_F / A_F
```

检查观测或模型 `Z_p(A)` 是否相对 UCD 呈合理电荷极化，而不是整体严重偏离。

- `WARN`：固定 `A` 的主要产额集中在离 UCD 很远的电荷区且无说明。
- `INFO`：偏离方向与已知电荷极化一致。
- `NA`：缺少 `Z_F`、`A_F` 或固定 `A` 电荷链。

### ODD-001：奇偶效应

低激发时偶 `Z`、偶 `N` 产额常增强，高激发时增强应减弱。

- `WARN`：低激发完整同位素分布完全无奇偶结构，或高激发仍有异常强奇偶振荡且无说明。
- `NA`：数据太稀疏、只给质量产额或能量未知。

### DECAY-001：累积产额衰变链逻辑

累积产额应包含母体衰变馈入，通常不能低于相同核素的独立产额，除非定义、截断或支路约定另有说明。

- `FAIL`：同一核素同一规范下 `CY < IY` 且差异远大于不确定度。
- `WARN`：累积产额沿衰变链出现明显断裂、支路遗漏或单位混用迹象。
- `NA`：无累积产额或无可比较独立产额。

### DECAY-002：不要累计求和误判

累积同位素产额求和会重复计数 beta 衰变链流量。

- `PASS`：报告避免把累积总和当成二碎片归一化。
- `FAIL`：数据说明声称完整累积同位素总和必须为 `2.0` 或 `200`，且据此做归一化。
- `NA`：非累积数据。

### ISO-001：同质异能态比例范围

同质异能态比例 `R_m = Y_m / (Y_g + Y_m)` 必须满足 `0 <= R_m <= 1`。

- `FAIL`：比例小于 0 或大于 1。
- `WARN`：基态和同质异能态产额单位或归一化疑似不一致。
- `NA`：无同质异能态信息。

### ISO-002：同质异能态物理趋势

同质异能态产额受自旋、宇称、能级密度和 gamma 级联影响。趋势判断应依赖核素具体能级结构，不要用统一阈值武断判错。

- `WARN`：相邻能量或相邻体系中同一核素比例突变且无不确定度或实验说明。
- `INFO`：比例变化可能与角动量或能级结构相关。
- `NA`：无可比较序列。

### PN-001：瞬发中子一致性

中子发射后产物质量分布相对于一次碎片质量分布应整体向低质量数移动，移动量与平均瞬发中子数相容。

- `WARN`：后中子分布被解释为前中子分布，或质量峰移动方向不合理。
- `INFO`：观察到合理的质量亏损或锯齿形瞬发中子结构。
- `NA`：无前/后中子对照或无瞬发中子信息。

### TKE-001：TKE 与瞬发中子关系

总动能较高通常对应可用于激发和中子蒸发的能量较少；TKE、TXE 和瞬发中子多重数之间应在趋势上相容。

- `WARN`：TKE 与瞬发中子多重数趋势强烈反向常识且无模型说明。
- `INFO`：趋势仅作为辅助证据。
- `NA`：无 TKE 或瞬发中子数据。

### LIB-001：本地评价库比较

如项目中有 ENDF/B、JEFF、JENDL、GEF 或本地评价文件，可比较相同裂变体系、能量、产额类型和归一化。

- `WARN`：与多个评价库在主峰、谷区或主产物上系统偏离且无说明。
- `INFO`：差异可能来自库版本、能量插值、独立/累积定义或前/后中子约定。
- `NA`：无可用参考库或规范不匹配。

## 常见误判

- 将部分测量产物集合当成完整分布。
- 将百分产额和每次裂变分数混用。
- 将一次碎片产额和中子发射后产物产额混用。
- 对高能、非锕系或奇异体系套用低能锕系双峰规则。
- 对累积产额直接求和并要求 `2.0` 或 `200`。
- 忽略多机会裂变造成的有效裂变核变化。
